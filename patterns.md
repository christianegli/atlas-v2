# Patterns

Promoted weekly from mistakes.md when 3+ similar failures share a root cause.
Each pattern has a mitigation strategy and tracks its source mistakes.

## Format

```
## [PATTERN NAME]
- **First observed:** [date]
- **Occurrences:** [count]
- **Description:** What keeps happening
- **Trigger conditions:** When this tends to appear
- **Mitigation:** What's being done about it
- **Status:** [active | resolved | promoted-to-principle]
- **Source mistakes:** [list of mistake dates/titles]
```
