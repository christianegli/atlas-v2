# Mistakes Log

**Append-only.** When corrected or when a task fails, log it here immediately.
The reflection engine promotes recurring mistakes to patterns.md weekly.

## Format

```
## YYYY-MM-DD HH:MM — [SHORT TITLE]
- **What happened:** Factual description of the failure
- **What was expected:** Correct behavior
- **Root cause:** Why it went wrong (retrieval miss? bad reasoning? hallucination?)
- **Category:** [retrieval | reasoning | style | task-completion | memory | other]
- **Severity:** [minor | moderate | major]
- **Fix applied:** What changed to prevent recurrence
- **Status:** [open | fixed | promoted-to-pattern]
```
